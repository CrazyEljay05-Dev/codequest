What is CSS?

CSS (Cascading Style Sheets) is what makes your web pages beautiful! It controls colors, fonts, layouts, and the overall design of your website. Without CSS, every page would look plain and unstyled.

We use CSS rules to style HTML elements. Each rule has a selector (which element to style) and a set of properties (what to change).

h1 {
  color: #ff7043;
  text-align: center;
  font-family: "Poppins", sans-serif;
}

p {
  color: #333;
  line-height: 1.6;
}
