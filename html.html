1. What is HTML?

HTML stands for HyperText Markup Language. It is the standard language for creating webpages and web applications. It describes the structure of a webpage using tags.

2. Structure of an HTML Page

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
  </body>
</html>
Output:

Hello World!

3. Basic Tags

  • <h1> - <h6> : Headings
  • <p> : Paragraph
  • <br> : Line Break
  • <hr> : Horizontal Rule
  • <strong> : Bold text
  • <em> : Italic text

4. HTML Attributes

Attributes provide additional information about HTML elements.

<a href="https://www.google.com">Visit Google</a>
Output: Visit Google

5. Lists

There are two main types of lists:

<ul>
  <li>Apple</li>
  <li>Banana</li>
</ul>

<ol>
  <li>First</li>
  <li>Second</li>
</ol>

6. Links and Images

<a href="page.html">Go to Page</a>
<img src="image.jpg" alt="Description">

7. Tables

<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Eljay</td>
    <td>20</td>
  </tr>
</table>

8. Forms

<form>
  Name: <input type="text"><br>
  Email: <input type="email"><br>
  <input type="submit" value="Submit">
</form>

9. Semantic Elements

Semantic tags clearly describe their meaning to both the browser and developer.

  • <header> - Header section
  • <nav> - Navigation links
  • <article> - Independent content
  • <footer> - Footer section

10. HTML5 Features

HTML5 introduced new elements and APIs such as:

  • <audio> and <video>
  • <canvas> for drawing
  • <section> and <aside>

11. Best Practices

  • Always use <!DOCTYPE html>.
  • Use proper indentation and nesting.
  • Include alt attributes in images.
  • Close all tags properly.
