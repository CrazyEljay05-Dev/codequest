What is HTML?

HTML (HyperText Markup Language) is the backbone of every website. It defines the structure and meaning of content on the web. HTML uses tags to organize and display text, images, links, tables, and more.

Basic Structure of an HTML Page

<!DOCTYPE html>
<html>
  <head>
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>Welcome to HTML learning with CodeQuest!</p>
  </body>
</html>

Hello, World!

Welcome to HTML learning with CodeQuest!

Common HTML Tags

  • <h1>–<h6> — Headings (from largest to smallest)
  • <p> — Paragraph text
  • <a> — Hyperlinks
  • <img> — Images
  • <div> — Generic container for grouping elements
  • <ul>, <ol>, <li> — Lists

HTML Attributes

Tags can have attributes that provide extra information. Attributes appear inside the opening tag and usually come in name/value pairs. 

<img src="logo.png" alt="CodeQuest Logo">

Why Learn HTML?

HTML is the **foundation** of every website. It defines *what* content appears on a page, while CSS controls how it looks, and JavaScript controls how it behaves. Understanding HTML is the first step to becoming a web developer.
