Introduction to HTML

HTML stands for HyperText Markup Language. It is the standard language used to create and design the structure of web pages. Every website you visit is built upon HTML as its foundation.

üìú Origin of HTML

HTML was invented in 1989 by Sir Tim Berners-Lee, a British scientist working at CERN (the European Organization for Nuclear Research). He designed HTML to share documents easily across computers connected by the Internet. The first version, HTML 1.0, appeared in 1991. The most widely used version today is HTML5, maintained by the WHATWG (Web Hypertext Application Technology Working Group).
