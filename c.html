What is C Language?

C is a powerful and general-purpose programming language developed to write operating systems, compilers, and embedded software. It provides low-level access to memory and offers excellent performance, which makes it one of the most important programming languages in computer science history.

Who is the Founder of C?

The C language was created by Dennis Ritchie at Bell Laboratories (AT&T) in the early 1970s. It evolved from an earlier language called B, which was developed by Ken Thompson.

When Did C Start?

C was officially released in 1972 and became widely used after the publication of the book ‚ÄúThe C Programming Language‚Äù by Brian Kernighan and Dennis Ritchie in 1978.

Why is C Important?

  • It is the foundation of many modern languages like C++, Java, and C#.
  • It provides low-level memory access for system programming.
  • Most operating systems, including UNIX and parts of Windows, were written in C.
  • It is portable ‚Äî code written in C can run on many types of computers.
  • It helps develop problem-solving and algorithmic thinking skills.

Without C Language Example

Before C, programming was done in assembly language or machine code, which looked like this:

MOV AX, 0002h
MOV BX, 0003h
ADD AX, BX
INT 21h

This is hard to read and understand for humans.

With C Language Example

#include <stdio.h>

int main() {
    int a = 2, b = 3, sum;
    sum = a + b;
    printf("Sum: %d", sum);
    return 0;
}
Output:
Sum: 5

Structure of a C Program

A C program usually has the following parts:

  • Header Files ‚Äì contain library functions (e.g., #include <stdio.h>).
  • main() Function ‚Äì entry point of every C program.
  • Variable Declarations ‚Äì where data is stored.
  • Statements and Expressions ‚Äì where logic and computation occur.
  • Return Statement ‚Äì ends the program.

Header Files

Header files give access to built-in functions. Common examples:

  • #include <stdio.h> ‚Äì standard input/output functions.
  • #include <math.h> ‚Äì mathematical functions.
  • #include <string.h> ‚Äì string manipulation.

Variables and Data Types

Variables store data, and data types define what kind of data they hold.

int age = 20;        // integer
float height = 5.9;  // decimal number
char grade = 'A';    // single character

Input and Output

#include <stdio.h>

    int age;
    printf("Enter your age: ");

    return 0;

Output:

You are 20 years old.

Operators in C

C uses operators to perform operations:

  • Arithmetic: + - * / %
  • Relational: == != < > <= >=
  • Logical: && || !
  • Assignment: = += -= *= /=

Conditional Statements

Conditions allow your program to make decisions:

#include <stdio.h>

int main() {
    int num = 10;
    if(num > 0)
        printf("Positive number");
    else
        printf("Negative number");
    return 0;
}

Loops in C

Loops help repeat tasks easily.

// Example of a for loop
#include <stdio.h>

int main() {
    for(int i = 1; i <= 5; i++) {
        printf("%d ", i);
    }
    return 0;
}
Output:
1 2 3 4 5

Functions in C

Functions divide programs into smaller parts, improving readability and reusability.

#include <stdio.h>

void greet() {
    printf("Hello, C Learner!");
}

int main() {
    greet();
    return 0;
}
Output:
Hello, C Learner!
